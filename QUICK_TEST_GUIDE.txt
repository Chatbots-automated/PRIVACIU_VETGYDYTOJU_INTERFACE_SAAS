## Common Issues & Solutions

Issue: "is_eco_farm" error still appears
Solution: Migration wasn't run. Go to Supabase SQL Editor and run the migration.

Issue: "komanda" plan not available in dropdown
Solution: Run migration 20260425000005 to add komanda to enum.

Issue: Subscription profile modal doesn't open
Solution: Check browser console. May need to clear cache (Ctrl+Shift+R).

Issue: TypeScript/build errors
Solution: Run `npm install` to ensure dependencies are up to date.
