REGISTRATION FLOW
================================================================================

Step 1: ADMIN CREATES CLIENT
-----------------------------
1. Login as client_admin
2. Go to /admin
3. Click "Add Client"
4. Fill in organization details
5. Choose subscription plan (trial/starter/professional/enterprise)
6. Click "Create Client"
7. REGISTRATION CODE IS GENERATED!

Example Code: ABCD-EFGH-IJKL

Step 2: ADMIN SENDS CODE TO CLIENT
-----------------------------------
Admin gets:
- Registration Code: ABCD-EFGH-IJKL
- Registration URL: http://yourapp.com/register?code=ABCD-EFGH-IJKL

Send via:
- Email
- SMS
- WhatsApp
- Or any communication method

Step 3: CLIENT REGISTERS
-------------------------
1. Client clicks "Registruotis" on login page
2. Enters registration code (XXXX-XXXX-XXXX)
3. System validates code automatically
4. Client sees organization info
5. Client can EDIT/ADD organization details:
   - Contact Person
   - Phone Number
   - Address
   - City
   - Postal Code
6. Client fills in PERSONAL account info:
   - Full Name (Vardas ir pavardė)
   - Email Address (El. paštas)
   - Password (Slaptažodis - min 8 chars)
   - Confirm Password
7. Client clicks "Sukurti paskyrą" (Create Account)
8. System updates client info if edited
9. Password is hashed with bcrypt
10. User account created and linked to client
11. Auto-login and redirect to app

Step 4: CLIENT USES SYSTEM
---------------------------
Client can now:
- Login with their email/password
- Access their organization's data
- Add more users (within limit)
- Manage farms
- Use all features based on plan
